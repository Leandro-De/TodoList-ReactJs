## Task App ReactJs

Task app made in ReactJs.

### What does the app do?

- You can make crud with each task.
- You can mark as complete each task.
- You can hide or show completed tasks.

### Development Details.

- CSS Bem methodology.
- Reack Hooks

## Installation

Install project with npm

```bash
  npm install
  cd project
```
